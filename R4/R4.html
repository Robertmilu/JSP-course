LESSON 13：テキストファイルを読み取ろう

  • 前回の連想配列の例ではデータの修正をするためには，プログラム自体を修正する必要がある．
  • データをプログラムから独立させるために，ファイルにデータを保存するようにする．

address3a.jsp

<%@ page contentType="text/html;charset=UTF-8"
	import="java.io.*, java.util.*"%>
<!DOCTYPE HTML>
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>住所録一覧</TITLE>
</HEAD>
<BODY>
	<TABLE BORDER="1">
		<TR>
			<TH>No.</TH>
			<TH>名前</TH>
			<TH>性別</TH>
			<TH>電話番号</TH>
			<TH>住所</TH>
		</TR>
		<%
			InputStreamReader reader = new InputStreamReader(
					new FileInputStream(application.getRealPath("/WEB-INF/data.txt")), "UTF-8");
			BufferedReader buf = new BufferedReader(reader);
			while (buf.ready()) {
				String line = buf.readLine();
				out.println("<TR>");
				//StringTokenizer token = new StringTokenizer(line, "\t");
				//while(token.hasMoreTokens()) {
				//	out.println("<td><font color=\"red\">" + token.nextToken() + "</font></td>");
				String[] token = line.split("\t");
				for (int i = 0; i < token.length; i++) {
					out.println("<TD><FONT COLOR=\"red\">" + token[i] + "</FONT></TD>");
				}
				out.println("</TR>");
			}
			buf.close();
		%>
	</>
</BODY>
</HTML>
  • ファイルは/home/student22/stuXXXX/public_html/report/WEB-INF/のフォルダに保存しておく．
  • application.getRealPath()メソッドを用いることで，WEB-INF以下の相対パスの記述だけでよくなる．
  • 教科書の例ではファイル読み取りのためにFileReaderクラスを用いているが，ここではFileInputStreamクラスを用いている．コンストラクタの第2引数に指定することで，文字コードの異なるファイルを扱うことが可能である．ちなみにサーバの文字コードはUTF-8，Windowsの文字コードはShift-JISである．教科書p.174参照．
  • BufferedReaderクラスでバッファを使ったファイルの読み取りが可能になり，処理が効率化される．
    • ready()メソッドでファイルが読み取り可能かどうかを確認する．
    • readLine()メソッドでファイルを1行読み取る．
    • close()メソッドで読み取りを終了する．
  • 文字の色を「赤」に変えるために，HTMLの<FONT COLOR="red">を用いている．これをout.println()メソッドで出力する場合，「"」がJavaの予約文字となっているために，エスケープシーケンス（\"）を用いている．教科書p.173参照．
  • StringTokenizerは非推奨なので，split()メソッドを用いている．

LESSON 14：カスタムログを記録しよう

FileWrite.jsp

<%@ page contentType="text/html;charset=UTF-8"
         import="java.io.*, java.util.*" %>
<!DOCTYPE HTML>
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>ファイルの書き込み</TITLE>
</HEAD>
<BODY>
<%
String path = "/WEB-INF/" + "test.txt";
FileWriter writer = new FileWriter(application.getRealPath(path), true);
BufferedWriter buf = new BufferedWriter(writer);
buf.write("2020/7/24,東京オリンピック");
buf.newLine();
buf.close();
%>
書き込み終了しました．
</BODY>
</HTML>
  • データは/home/student22/stuXXXX/public_html/report/WEB-INF/test.txtに保存される．
  • ファイル書き込みのためにFileWriterクラスを用いている．コンストラクタの第2引数をtrueにすることで，追加の書き込みをおこなう．trueにしないと，ファイルが上書きされ，元の内容は消去される．
  • BufferedWriterクラスでバッファを使ったファイルの読み取りが可能になり，処理が効率化される．
    • write(string)メソッドでstringをファイルに書き込む．．
    • newLine()メソッドで改行を書き込む．
    • close()メソッドで書き込みを終了する．
  • ファイル書き込みが拒否される場合は，WinSCPでそのファイルを選択し，右クリックする．「プロパティ」を選択し，パーミッションの「その他」の「W」にチェックを入れる．FileZillaではそのファイルを選択し，右クリックする．「パーミッションの変更」を選択し，「その他のパーミッション」の「書き込み」にチェックを入れる．

LESSON 15：正規表現検索に挑戦しよう

入力フォーム(address4Ain.jsp)

<%@ page contentType="text/html;charset=UTF-8" %>
<!DOCTYPE HTML>
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>アドレス帳検索</TITLE>
</HEAD>
<BODY>
<FORM METHOD="POST" ACTION="address4Aout.jsp">
<LABEL>検索する電話番号：
  <INPUT TYPE="text" NAME="number" SIZE="20" />
</LABEL>
<INPUT TYPE="submit" VALUE"検索" />
</FORM>
</BODY>
</HTML>

結果ページ(address4Aout.jsp)

<%@ page contentType="text/html;charset=UTF-8"
         import="java.io.*, java.util.*, java.util.regex.*" %>
<!DOCTYPE HTML>
 
<%
String pnum = "";
 
request.setCharacterEncoding("UTF-8");
String num=request.getParameter("number");
if(num == "") pnum=".*"; else pnum=num;
Pattern pattern = Pattern.compile(pnum);
%>
 
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>住所録一覧</TITLE>
</HEAD>
<BODY>
正規表現文字列：<%= pnum %>
<TABLE BORDER="1">
<TR>
	<TH>No.</TH><TH>名前</TH><TH>性別</TH><TH>電話番号</TH><TH>住所</TH>
</TR>
<%
InputStreamReader reader = new InputStreamReader( new FileInputStream(
	application.getRealPath("/WEB-INF/data.txt")), "UTF-8");
BufferedReader buf = new BufferedReader(reader);
while(buf.ready()) {
	String line = buf.readLine();
	String[] token = line.split("\t");
	String no = token[0];
	String name = token[1];
	String sex = token[2];
	String phone = token[3];
	String address = token[4];
 
	Matcher match = pattern.matcher(phone);
	if(match.find()) {
		out.println("<TR>");
		out.println("<TD>"+no+"</TD>");
		out.println("<TD>"+name+"</TD>");
		out.println("<TD>"+sex+"</TD>");
		out.println("<TD>"+phone+"</TD>");
		out.println("<TD>"+address+"</TD>");
		out.println("</TR>");
		}
	else continue;
}
buf.close();
%>
</TABLE>
</BODY>
</HTML>
  • 正規表現に関しては教科書の解説(p.189~)やサルにもわかる正規表現入門などを参考にすること．
  • Pattern.compile(正規表現文字列)で正規表現パターンを生成する．Patternクラスを利用するにはjava.util.regex.*をインポートする必要がある．
  • matcher(検索文字列)で正規表現による検索を行う．
  • find()がtrueであれば，検索がヒットしたことが分かる．

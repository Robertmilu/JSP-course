LESSON 6：クエリー情報を活用しよう

入力フォーム(input4.jsp)

<%@ page contentType="text/html; charset=UTF-8" %>
<!DOCTYPE HTML>
<HTML><HEAD>
<META CHARSET="UTF-8">
<TITLE>入力画面</TITLE>
</HEAD>
<FORM ACTION="query.jsp" METHOD="Get">
入力画面<BR>
<TABLE>
<TR><TD>氏名</TD><TD><INPUT TYPE="text" NAME="name"></TD></TR>
<TR><TD>年齢</TD><TD><INPUT TYPE="text" NAME="age" VALUE="21"></TD></TR>
</TABLE>
<INPUT TYPE="submit" VALUE="行け！"><BR>
</FORM>
</HTML>
  • LESSON4のinput3.jspと異なっているのは，METHOD="Get"になっているところ．JSPでフォーム情報をサーバに送るには，GetメソッドとPostメソッドがある．
  • Getメソッドではクエリ情報（フォーム情報）がURLに含まれて送られる． 例：http://dius01.ksc.kwansei.ac.jp/network/students09/stu9999/report/query.jsp?name=kitamura&age=55
  • Postメソッドではフォーム情報はURLに含まれない．
  • INPUTタグのパラメータにVALUEを加えると初期値が設定できる．

結果ページ(query.jsp)

<%@ page contentType="text/html;charset=UTF-8"
         import="java.util.*" %>
<!DOCTYPE HTML>
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>クエリ情報を取得</TITLE>
</HEAD>
<BODY>
<TABLE BORDER="1">
<%
Enumeration params = request.getParameterNames();
while(params.hasMoreElements()) {
	String name = (String)(params.nextElement());
	out.println("<tr><th>" + name + "</th>");
	out.println("<td>" + request.getParameter(name) + "</td></tr>");
}
%>
</TABLE>
</BODY>
</HTML>
  • クエリ情報が2バイト文字（日本語の全角文字）の場合は文字化けするので注意．
  • request.getParameterNames()でクエリ情報を一括して獲得する．
  • hasMoreElements()でEnumerationオブジェクトにデータが残っているかどうかをチェックする．
  • nextElement()でEnumerationオブジェクトからデータを順に取り出す．
  • request.getParameter(name)でクエリ情報のキー名nameの値を得る．

LESSON 8：クッキー情報を活用しよう

cookie.jsp

<%@ page contentType="text/html; charset=UTF-8" %>
<%
boolean flag = false;
int count = 1;
Cookie cook = null;
Cookie[] cookies = request.getCookies(); //クッキー情報を取り出す
if(cookies != null) { //クッキー情報が空でなければ
	for(Cookie current : cookies) { //クッキー情報を1つずつ処理
		if(current.getName().equals("cnt") == true) {
			count = Integer.parseInt(current.getValue()) + 1;
			cook = new Cookie("cnt", (new Integer(count)).toString());
			//値を1加えた新しいクッキーを作る
			cook.setMaxAge(60 * 60 * 24 * 180);
			response.addCookie(cook);
			flag = true;
			break;
		}
	}
}
if(!flag) { //クッキーを食べたことがなければ
	cook = new Cookie("cnt", "1");
	cook.setMaxAge(60 * 60 * 24 * 180);
	response.addCookie(cook);
}
%>
<!DOCTYPE HTML>
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>クッキーで簡易アクセスカウンター</TITLE>
</HEAD>
<BODY>
あなたはこのサイトに<%=count %>回、アクセスしました。
</BODY>
</HTML>
  • request.getCookiers()でクッキー情報を一括して獲得する．
  • getName()でクッキーの名前を取り出す．
  • getValue()でクッキーの値を取り出す．
  • setMaxAgeでクッキーの有効期限を設定する．
  • response.addCookie(cook)でクッキーcookをブラウザに与える．
  • for(Cookie current : cookies){...}
    は
    for(int i=0; i<cookies.length; i++) {Cookie current=cookies[i]; ...}
    と同じ意味である．

	response.addCookie(cookie);
	flag = true;
}
 
//クッキーの処理
Cookie[] cookies = request.getCookies();
if(cookies != null) {
	for(int i = 0; i < cookies.length; i++){
		if(cookies[i].getName().equals("UserName")) { 
			message = URLDecoder.decode(cookies[i].getValue(),"UTF-8");
			flag = true;
		}
	}
}
%>

<HEAD>
<META CHARSET="UTF-8" />
<TITLE>Cookieのチェック</TITLE>
</HEAD>
<BODY>
<BR>
<% if(!flag) { %>
<FORM ACTION="cookie1.jsp" METHOD="Post"> <!-- 自分自身に飛ぶ -->
入力画面<BR>

  • 参考：Cookie（クッキー）使用同意を求めるポップアップについて

クッキー情報のリセット

  • Chromeの場合：「設定」→「プライバシーとセキュリティ：閲覧履歴データの削除」→「Cookieと他のサイトデータ」にチェック→「データを削除」
  • Firefoxの場合：「Firefox」→「オプション」→「プライバシー」→「Cookieを個別に削除」→「すべてのCookieを削除」
  • Internet Explorerの場合：「ツール」→「インターネットオプション」→「全般」→「閲覧の履歴」→「削除」→「Cookie」にチェックを入れる，「お気に入りWebサイトデータを保持する」のチェックを外す→「削除」
  • ブラウザを再起動しないとリセットされないこともある．

LESSON 9：セッション情報を活用しよう

session.jsp

<%@ page contentType="text/html; charset=UTF-8" %>
<%
Integer count = (Integer)session.getAttribute("cnt");
if(count == null) {
	session.setAttribute("cnt", 1);
	count = 1;
} else {
	count++;
	session.setAttribute("cnt", count);
}
%>
<!DOCTYPE html>
<HTML>
<HEAD>
<META CHARSET="UTF-8" />
<TITLE>セッションで簡易アクセスカウンター</TITLE>
</HEAD>
<BODY>
あなたはこのサイトに<%=count %>回、アクセスしました。
</BODY>
</HTML>
  • session.getAttribute(name)でnameというキー名のセッション情報を取り出す．
  • session.setAttribute(name, value)でnameというキー名のセッション情報にvalueという値を設定する．
  • セッション情報はブラウザを閉じるとリセットされる．クッキー情報はブラウザを閉じてもリセットされない．
  • クッキーは個人情報を獲得する手段として利用される．
  • Integerはint型のラッパークラスでを整数値をオブジェクトとして扱いたい時に利用する．
